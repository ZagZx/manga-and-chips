Observações

O filtro de conteúdos de usuário ainda não é funcional, as configurações apenas são adicionadas no banco e exibidas na página de configurações do usuário, mas por padrão só é possível pesquisar por mangás seguros e sugestivos

Se alguma rota travar por muito tempo tente recarregar a página


Como usar o site

No menu à direita da barra de pesquisa é possível cadastrar e logar usuário, caso já esteja logado é possível ver as configurações do usuário para filtros de conteúdo, trocar senha e deletar conta


É possível pesquisar um mangá por nome na barra de pesquisa e adicioná-lo a biblioteca(se estiver logado)


Acessando a página de biblioteca é possível visualizar e remover os mangás que foram adicionados anteriormente


Em breve

Otimização de como a biblioteca funciona | quanto mais mangás são adicionados a biblioteca mais lenta a página fica pois toda vez que for carregar faz a requisição de cada mangá na API e só então carrega a página, será implementado um lazy loading
